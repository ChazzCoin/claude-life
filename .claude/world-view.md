# World-view

How I see the world. The descriptive frame. Different from
values (how I want to move) and vision (where I'm going) —
this is how I think reality, society, humanity, technology
actually *work*.

> Edit when something fundamental shifts in how I understand
> the world. Mostly stable across years; can shift in
> recognizable arcs (a year of reading, a hard experience, a
> relationship). Use `/decision` and archive the prior version
> when a real shift happens.
